#scripts
This repo contains scripts that I've found useful over the years

## ChromeApp

Creates an app using google chrome with it's own profile, great alternative to fluid as it enables extensions.

``` 
    1.  ./ChromeApp.sh
    2.  Enter app name
    3.  Enter url
    4.  Enter icon full path
```
